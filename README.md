# Author website of Erin Price
This website was built from a Jekyll theme called [Ice bear](http://jekyllthemes.org/themes/ice-bear/). It was then modified to be an author webpage.

To use this theme, you will need to download Ruby to test and serve it on your computer.

The Jekyll installation instructions are found [here](https://jekyllrb.com/docs/).
Ruby installation instructions are found [here](https://www.ruby-lang.org/en/downloads/).
Ruby depends on GCC. This website was helpful for me to [install it](https://preshing.com/20141108/how-to-install-the-latest-gcc-on-windows/).
